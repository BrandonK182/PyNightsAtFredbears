# PyNightsAtFred's
A PyGame recreation of the popular Five Nights at Freddy's game featuring legally distinct enemies
- Fred
- Bunnie
- Chick
- Vixen

Current Features
- Map layout reminiscent of game
- Enemies moving around the map
